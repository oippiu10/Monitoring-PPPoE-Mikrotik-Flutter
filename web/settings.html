Konfigurasi Router Mikrotik

Router Terdaftar
Panduan Koneksi Mikrotik

Prasyarat agar bisa terhubung:

  1. Buka web panel via XAMPP/Laragon (bukan Live Server)
  2. Di Mikrotik: IP > Services → aktifkan api (port 8728)
  3. Pastikan IP server ada di Allowed Address API service
  4. Firewall Mikrotik bolehkan port 8728 dari IP server
  5. Username & password harus punya hak akses API di Mikrotik
