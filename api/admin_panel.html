🔔 Trigger Notifikasi Update

Catatan: Sistem ini menggunakan Local Notifications, bukan Push Notifications. Notifikasi akan muncul saat user membuka aplikasi atau saat background check berjalan.

ℹ️ Cara Kerja:
1. Pastikan sudah upload APK baru ke hosting
2. Update version_config.php dengan versi baru
3. Klik tombol di atas untuk trigger notifikasi
4. User akan menerima notifikasi saat membuka aplikasi atau background check berjalan
